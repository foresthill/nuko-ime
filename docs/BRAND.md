# ぬこIME ブランド・ビジュアルアイデンティティ メモ

ぬこIME のブランディング・アイコン・キャラクター方針のラフメモ。
確定方針ではなく、後で深掘りする際の出発点。

最終更新: 2026-04-25

---

## 名前の由来

「ぬこ」は 2ch (現 5ch) で広まった「猫」の俗語。
2ch 文化への敬意から名付けたが、ぬこIME は 2ch / 5ch とは無関係。

## ビジュアル方針 (案)

### 推奨: オリジナル「ぬこIMEくん」キャラを作る
- 2ch AA 文化を**リスペクト**するスタイル (脱力系、丸み、シンプル造形)
- ただし**オリジナル創作** → 著作権 100% 自前、堂々と使える
- アプリアイコン、起動スプラッシュ、ドキュメントの要所に配置

### 文化的リファレンスとして AA を散りばめる (低リスク)
- README 冒頭にぬこ AA
- ターミナル起動時の小さな AA
- リリースノートで時々モナー風コメント
- AA キャラ (モナー・しぃ等) は判例上 **準・パブリックドメイン**扱い、安心して使える

### 避けるべきこと
- 既知の 2ch 公式デザイン要素のコピー
- 5ch の現行ロゴと紛らわしい配色・タイポ
- AA キャラを「ぬこIME 独自のもの」として商標主張する動き
- 出自を曖昧にした使用 (リスペクトを欠いた利用に映る)

## 著作権・商標の論点

### 「ぬこ」自体
- 単語1個に著作権は発生しない (日本著作権法2条1項1号)
- 商標は要確認: J-PlatPat で「ぬこ」「nuko」を**第9類 (ソフトウェア)** で検索
- 検索でクリーンなら使用OK

### 2ch AA キャラの法的状況 (参考)
- **モナー事件 (2002)** で avex の「のまネコ」騒動 → モナーは公共財扱いが定着
- Wikipedia 日本版も AA キャラを自由に掲載
- 商業独占的利用が炎上の引き金、非商業 OSS なら問題視されにくい

## コミュニティ取り込み戦略 (案)

| 手段 | 効果 |
|------|------|
| 5ch ソフトウェア板で告知 | 興味層に直接到達 |
| X (旧Twitter) / ニコニコ動画 で AA 風プレゼン | 文化的親和性で口コミ |
| VISION.md / BRAND.md で「2ch 文化へのリスペクト」を明示 | スタンスの誠実さ |
| AA で書かれた README やチュートリアル | 文化継承の自然な形 |
| 「AA 描ける人募集」を CONTRIBUTING に明記 | コミュニティから AA 資産が集まる |

**スタンス**: 文化を「消費しに行く」のではなく、**敬意を表しつつ独自の場を作る**。
